# 📊 Time Series Forecasting using Machine Learning

## 📌 Overview
This project focuses on predicting future stock prices using machine learning techniques. Time series forecasting is widely used in finance, energy, and weather prediction.

In this project, I used historical stock data (Tesla) to build predictive models and evaluate their performance.

---

## 🎯 Objectives
- Analyze time series data
- Build predictive models using Machine Learning
- Compare predicted vs actual values
- Evaluate model performance

---

## 🧠 Models Used
- Linear Regression (baseline model)
- Random Forest Regressor (improved model)

---

## 📂 Dataset
- Source: Yahoo Finance (via yfinance)
- Stock: Tesla (TSLA)
- Time Period: 2020–2024

---

## ⚙️ Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- yfinance

---

## 📈 Results
The Random Forest model outperformed Linear Regression in capturing non-linear patterns in stock price movement.

---

## 📊 Output
The model generates a plot comparing actual vs predicted stock prices.
